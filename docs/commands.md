# Commands

## User Commands

- `/request_movie <title>` — Request a movie by title.
- `/request_tv <title>` — Request a TV show by title.
- `/report_issue <title>` — Report an issue with a media item.
- `/quota_status` — Check your current quota (if enabled).

## Admin Actions

- Approve/Decline requests via buttons in Discord.
- Comment on or close issues via Discord.

## Example Usage

```
/request_movie Interstellar
/request_tv Breaking Bad
/report_issue The Office
/quota_status
```

## Notes

- Some commands may be disabled by configuration.
- Admin actions are only available to users with the correct Discord roles.
