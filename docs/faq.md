# FAQ

## What is Approvarr?
A Discord bot for managing Overseerr/Jellyseerr requests and issues directly from Discord.

## How do I install Approvarr?
See the [Installation](Installation.md) guide for Docker and manual setup instructions.

## How do I configure Approvarr?
See the [Configuration](configuration.md) guide for environment variables and Discord setup.

## How do I request media?
Use the `/request_movie` or `/request_tv` commands in Discord.

## How do I report an issue?
Use the `/report_issue` command in Discord.

## How do I get support?
Join the [Discord server](https://discord.gg/MPwvd9re) or open a [GitHub Issue](https://github.com/nicholg90/approvarr/issues).

## Can I contribute?
Yes! See the [Contributing](contributing.md) guide.
